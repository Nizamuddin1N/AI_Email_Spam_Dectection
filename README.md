## 📌 Overview
Built a machine learning system to classify messages as **Spam or Not Spam** using NLP techniques and efficient probabilistic modeling.

---

## ⚡ Highlights
- Implemented **TF-IDF feature extraction**
- Used **Naive Bayes** for fast classification
- Real-time **prediction interface**
- Lightweight and efficient model

---

## 🧠 Tech Stack
- Python
- Scikit-learn
- TF-IDF Vectorizer
- NLP preprocessing

---

## 🔄 Workflow
Input Text → Preprocessing → TF-IDF → Model Prediction → Output (Spam/Not Spam)
